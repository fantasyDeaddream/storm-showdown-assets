指挥官平衡性改动 · Commander Changes

诺娃 Nova

  • 防御无人机护盾持续时间：30S【21S】 → 14S【10S】（同DS）
21S的持续时间对于第一波压制有着过于优秀的作用，一个防御无人机几乎可以化解任何第一波的反压制。调整到10S（同DS），并不影响打团——大多数团战的接触都在10S内完成，玩家对防御无人机的释放需要更加精细。
  • 隐秘女妖火箭弹幕 CD：15S【10.7S】 → 28S【20S】（同DS）
成型的隐秘女妖对于地面单位和农民有毁灭性的杀伤能力，将其增加到DS的CD以减弱该能力。
  • 快速投送增加间隔 CD：2S【1.4S】（同霍纳）

蒙斯克 Mengsk

  • 元首鬼影迷宫式隐形迷彩持续时间：10S【7.1S】 → 6.5S【4.6S】
  • 焰能爆燃能量消耗提升到 100，CD：5S【4S】 → 14S【10S】
元首鬼影的综合能力在大多时间过于优秀，数值改动到同DS。
  • 帝国仲裁机血量：回调到原版 150
空投壁垒卫士已成为蒙斯克的主流打法，300+100血量追击空投的难度过大。
  • 释放虫群的价格、数量、能量消耗减半

沃拉尊 Vorazun

  • 黑暗圣堂武士暗影之怒 CD：15S【11S】 → 11S【8S】
  • 暗影之怒伤害：15（+20 对轻甲） → 20（+15 对轻甲）
  • 闪现 CD：10S【7S】 → 20S【14S】
部分同DS（DS中闪现为28S【20S】CD，但对战来说过长）。该改动旨在提升黑暗圣堂武士的正面作战能力，减弱控制与机动能力。

斯托科夫 Stukov

  • 腐蚀粘液科技研发时间：降低到 100S【71S】
  • 虫巢女王增强型线粒体能量回复速度加成：100% → 30%（同DS）
  • 天赋 "瘟疫守望者"：女妖发射感染人间隔 0.2 → 0.1

德哈卡 Dehaka

  • 穆尔瓦孵化虫群技能 CD：降低到 10S【7S】
  • 暴龙兽穿刺打击晕眩时间：2S【1.4S】 → 1.4S【1S】（同DS）

凯瑞甘 Kerrigan

  • 修复刺蛇攻击伤害和射程 Bug
  • 欧米茄虫道建造时间：增加到 10S【7S】
